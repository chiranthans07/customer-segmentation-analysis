# Customer Segmentation using K-Means Clustering

## Overview
This project performs customer segmentation using machine learning techniques. The goal is to group customers based on their purchasing behavior to identify different customer segments.

## Objective
The objective of this project is to analyze customer data and segment customers into distinct groups using the K-Means clustering algorithm.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Dataset
The dataset contains customer information including:
- Customer ID
- Gender
- Age
- Annual Income (k$)
- Spending Score (1-100)

## Steps Performed
1. Data loading and exploration
2. Feature selection (Income and Spending Score)
3. Determining optimal clusters using Elbow Method
4. Applying K-Means clustering algorithm
5. Visualizing customer segments

## Visualizations
- Elbow Method graph to find optimal clusters
- Scatter plot showing customer segments and centroids

## Key Insights
- Customers can be grouped into distinct segments based on spending behavior.
- Some customers have high income but low spending.
- Some customers have low income but high spending.
- These segments help businesses target customers more effectively.

## Conclusion
Customer segmentation helps businesses understand different customer groups and develop targeted marketing strategies. This project demonstrates how machine learning can be used for business insights.
